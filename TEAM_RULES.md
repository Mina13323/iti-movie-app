# Team Rules

## Do not change
- folder structure
- constants names
- route paths
- store names
- env variable names

## Shared naming
- useAuthStore
- useWishlistStore
- useMovieFilterStore
- authUser
- isAuthenticated
- wishlistItems
- selectedGenres
- selectedSort
- currentLanguage
- themeMode

## Branches
- main = production only
- develop = integration branch
- feature/auth-state
- feature/discovery
- feature/details-routing
- feature/wishlist-ui

## Rules
- no direct push to main
- no direct push to develop
- open PR to develop
- commit daily
- do not edit another member's scope without agreement
